# Content Lab Kit

**Free, high-signal tools for creators who ship.**

Content Lab Kit is an open product built in public. It helps creators move from a blank page to a clearer, more intentional post without generic AI sludge.

## Currently shipping

- **Hook Lab** — generate and refine distinct hook directions by platform and voice.
- **CTA Library** — coming next.
- **Caption Check** — in the lab.
- **Format Vault** — in the lab.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build in public

Progress is documented in [`CHANGELOG.md`](CHANGELOG.md). Small, useful improvements ship regularly.

## Principles

1. Useful before clever.
2. A clear point of view over infinite options.
3. Fast enough to stay in the creative flow.
4. Free tools should still feel excellent.
